# Horror Game (Unity 3D)

Horror Game adalah game bergenre horor 3D yang dikembangkan menggunakan **Unity Engine**.  
Project ini bertujuan untuk menghadirkan pengalaman bermain dengan suasana mencekam, eksplorasi lingkungan, dan elemen jumpscare.
